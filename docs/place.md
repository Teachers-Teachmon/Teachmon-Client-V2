# 🚀 BSSM 학생 일정 및 장소 상태 API Specification

본 문서는 **학생 일정 및 각 층별 장소 상태** 조회를 위한 공식 API 명세서입니다.

---

## 🏛️ 장소 및 상태 조회

### 1. 모든 층 상태 조회
- **Method**: `GET`
- **Path**: `/student-schedule/place`
- **Description**: 각 층(1층~4층)에 현재 이석 또는 자습 상태인 교실이 각각 몇 곳인지 조회합니다.
- **Response Format**: `[1층 교실 수, 2층 교실 수, 3층 교실 수, 4층 교실 수]`
- **Response Example**:
```json
[
  2, 
  4, 
  0, 
  2
]
```
> [!NOTE]
> 응답 배열의 인덱스는 층을 의미합니다 (0: 1층, 1: 2층, 2: 3층, 3: 4층).

> 이전 버전의 감독 순위, 분기 관리, 이탈 기록 관련 API는 현재 노션에서 제거되었으며, 본 명세서에서도 제외되었습니다.
